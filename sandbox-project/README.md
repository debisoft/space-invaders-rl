# Space Invaders DeepRL Agent

A retro Space Invaders game featuring a Deep Reinforcement Learning (DQN) agent that learns to play the game in real-time.

## Features
- **Classic Gameplay**: Defeat waves of invaders and a challenging Boss.
- **DeepRL Agent**: A TensorFlow.js powered agent that observes the game state and learns to maximize score.
- **Cyberpunk Aesthetic**: Premium UI with CRT effects, neon visuals, and glassmorphism.
- **Cloud Models**: Save and load trained models via Supabase.

## Controls
- **Arrow Keys**: Move
- **Space**: Shoot / Start Game
- **UI Panel**: Train, Save, Load models.

## Tech Stack
- Vanilla JS, HTML5 Canvas, CSS3
- TensorFlow.js
- Supabase
